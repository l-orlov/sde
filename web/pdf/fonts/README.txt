Blinker font for mPDF (clasico_company_es/en, clasico_pdf_es/en).

Google Fonts (all weights 100–900):
  https://fonts.googleapis.com/css2?family=Blinker:wght@100;200;300;400;600;700;800;900&display=swap

For PDF generation mPDF needs local TTF files. Download the font:
  1. Open https://fonts.google.com/specimen/Blinker
  2. Click "Download family"
  3. Unzip and copy into this folder (web/pdf/fonts/):
     - Blinker-Regular.ttf  (or Blinker-Regular.ttf from the zip)
     - Blinker-Bold.ttf     (or Blinker-Bold.ttf from the zip)

Alternative: https://github.com/google/fonts/tree/main/ofl/blinker
